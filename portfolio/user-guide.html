Getting Started with QuickSync

Connect your first workspace and sync your first file in under five minutes.

Audience: New users, no technical background assumed
Product: QuickSync (fictional file-sync product, used for portfolio purposes)
Status: Published
Last updated: March 2026

Before you start

You’ll need a QuickSync account and access to the folder you want to sync. This guide covers the desktop app on Windows and macOS — mobile setup is covered separately.

Time required About 5 minutes, plus the time it takes your first sync to finish in the background.

1. Create your workspace

  1. Open QuickSync and select Create a workspace.
  2. Name the workspace something your team will recognize — for example, the project or department name.
  3. Choose Private or Shared. You can change this later in Workspace settings.

2. Connect a folder

  1. From the workspace dashboard, select Add folder.
  2. Choose the folder on your computer you want to keep in sync.
  3. QuickSync scans the folder and shows an estimated upload time before anything starts.
Before you continue Folders larger than 50 GB sync in the background and may take several hours on the first pass. You can keep working — QuickSync won’t lock the folder.

3. Confirm your first sync

Once the folder is added, QuickSync shows a progress bar in the menu bar (macOS) or system tray (Windows). When it turns green, your files are fully synced and any teammate added to the workspace can see them.

Troubleshooting

What you’re seeingLikely causeWhat to do
Sync stuck at the same percentage for 10+ minutesA single large file is in progressCheck Activity → Current file to confirm it’s still moving
Folder shows a yellow warning iconA file is open in another appClose the file and QuickSync retries automatically
“Workspace full” messagePlan storage limit reachedFree up space or upgrade your plan from Account → Billing
